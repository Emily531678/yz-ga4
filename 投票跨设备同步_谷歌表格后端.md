# 投票跨设备同步（手机/电脑票数合并）

你现在页面的投票统计用的是 `localStorage`，所以手机和电脑不会共享数据。要合并票数，需要一个“云端存储”。最容易免费实现的方式是：Google Sheets + Apps Script（当作一个简单 API）。

## 1. 新建 Google Sheet

1. 打开 Google Sheets，新建一个空表格
2. 重命名表格：`yz-ga4-votes`
3. 新建一个工作表（Sheet）名字：`votes`
4. 在 `votes` 里填表头：
   - A1：`place`
   - B1：`count`
5. 在 A2-A7 依次填 6 个景点（与网页一致）：
   - 瘦西湖
   - 个园
   - 何园
   - 大明寺
   - 东关街
   - 古运河
6. B2-B7 都填 `0`

## 2. 建 Apps Script（生成 API）

1. 在表格里：Extensions（扩展程序）→ Apps Script
2. 删除默认代码，粘贴下面代码

```javascript
function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('votes');
  var payload = {};

  try {
    payload = JSON.parse(e.postData.contents || '{}');
  } catch (err) {
    payload = {};
  }

  var action = payload.action;
  if (action === 'get') {
    return json_(getVotes_(sheet));
  }

  if (action === 'vote') {
    var place = String(payload.place || '').trim();
    if (!place) return json_({ error: 'place_required' }, 400);
    return json_(addVote_(sheet, place));
  }

  return json_({ error: 'bad_request' }, 400);
}

function getVotes_(sheet) {
  var values = sheet.getDataRange().getValues();
  var votes = {};
  for (var i = 1; i < values.length; i++) {
    var place = String(values[i][0] || '').trim();
    var count = Number(values[i][1] || 0);
    if (place) votes[place] = count;
  }
  return { votes: votes };
}

function addVote_(sheet, place) {
  var values = sheet.getDataRange().getValues();
  for (var i = 1; i < values.length; i++) {
    var p = String(values[i][0] || '').trim();
    if (p === place) {
      var current = Number(values[i][1] || 0);
      sheet.getRange(i + 1, 2).setValue(current + 1);
      return getVotes_(sheet);
    }
  }
  var lastRow = sheet.getLastRow();
  sheet.getRange(lastRow + 1, 1).setValue(place);
  sheet.getRange(lastRow + 1, 2).setValue(1);
  return getVotes_(sheet);
}

function json_(obj, status) {
  var output = ContentService.createTextOutput(JSON.stringify(obj));
  output.setMimeType(ContentService.MimeType.JSON);
  return output;
}
```

## 3. 部署成 Web App

1. 右上角 Deploy → New deployment
2. Select type：Web app
3. Execute as：Me
4. Who has access：Anyone
5. Deploy
6. 复制 Web app URL（形如 `https://script.google.com/macros/s/.../exec`）

## 4. 把 URL 填进网页（让手机和电脑共享票数）

打开本项目的 [index.html](file:///d:/vtc/%E7%BD%91%E9%A1%B5%E4%BD%9C%E4%B8%9A/yz/index.html) 找到这一行：

- `const VOTE_BACKEND_URL = '';`

把空字符串改成你的 Web app URL，例如：

- `const VOTE_BACKEND_URL = 'https://script.google.com/macros/s/XXXX/exec';`

然后提交并推送到 GitHub：

```bash
git add index.html
git commit -m "Enable vote backend sync"
git push
```

2. 用手机打开同一个网址（再投一票）
